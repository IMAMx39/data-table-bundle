# StringFilter (Doctrine ORM)

The [StringFilter](../../../../src/Bridge/Doctrine/Orm/Filter/StringFilter.php) represents a filter that operates on string values.

## Supported operators

- `Operator::EQUAL`
- `Operator::NOT_EQUAL`
- `Operator::CONTAINS`
- `Operator::NOT_CONTAINS`

## Options

This filter has no additional options.
