# BooleanFilterType

The [BooleanFilterType](https://github.com/Kreyu/data-table-bundle/blob/main/src/Bridge/Doctrine/Orm/Filter/Type/BooleanFilterType.php) represents a filter that operates on boolean values.

## Supported operators

- `Operator::EQUALS`
- `Operator::NOT_EQUALS`
